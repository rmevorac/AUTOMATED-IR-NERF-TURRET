Detailed Description

This class implements a simple driver for MMA8451 and MMA8452 accelerometers.

These inexpensive phone accelerometers talk to the CPU over I2C. Only basic functionality is supported:

  • The device can be switched from standby mode to active mode and back
  • Readings from all three axes can be taken in A/D bits or in g's
  • The range can be set to +/-2g, +/-4g, or +/-8g There are many other functions supported by the accelerometers which could be added by someone with too much time on her or his hands :P

An example of how to use this driver:

mma = mma845x.MMA845x (pyb.I2C (1, pyb.I2C.MASTER, baudrate = 100000), 29)
mma.active ()
mma.get_accels ()

def mma845x.MMA845x.active (   self)

Put the MMA845x into active mode so that it takes data.

In active mode, the accelerometer's settings can't be messed with. Active mode is set by setting the ACTIVE bit in register CTRL_REG1 to one.
